{% trans "Парсеры сайтов" %}

{% blocktrans %} Парсинг ( теже скрипты) только их цель собрать/отфильтровать информацию в большом объёме с сайта и предоставить в удобоваримом виде для человека. Будь то csv или xml файл. Тимон использую при парсинге beatufulsoup ( и selenium при необходимости рендерить страницу). Хотя идеальный парсер это тот который использует API сайтов и получает информацию напрямую с сайта.
